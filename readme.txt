Simple Employee Management Program

Description
This C++ program manages employee data, including employee number, name, and salary, for a predefined number of employees.

Features
-   Takes employee number, name, and salary as input for each employee
-   Stores and displays employee information in a structured format.
-   Uses a class to represent employee data.
-   Allows setting and retrieving employee details.

Code Structure
The code consists of three files:
-   main.cpp: Contains the main function with the program logic to take employee inputs and display employee details.
-   header.h:  Contains the declaration of the `Employee` class.
-   implementation.cpp: Contains the implementation of the `Employee` class member functions.

Dependencies
-   iostream: For input/output operations.
-   string: For handling employee names.

Compilation and Execution
To compile and run the program, save the code in the respective files (main.cpp, header.h, and implementation.cpp) and use a C++ compiler.
